# Project with Virtual Environment set up with virtualenv

## Requirements

- Python 3.11
- virtualenv 20.28.0

## Installation

```bash
virtualenv venv -p python3.11
```

## Activation

```bash
source venv/bin/activate
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## Deactivation

```bash
deactivate
```
