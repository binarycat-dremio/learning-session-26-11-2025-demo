# Project with Virtual Environment set up with Poetry

## Requirements

- Python 3.11
- Poetry >=2.1.0

## Installation

```bash
poetry install
```

## Activation/Deactivation (Optional)

```bash
poetry env activate
deactivate
```

## Install dependencies

```bash
poetry add <package_name>
```

## Usage
The activation/deactivation is optional. 
You can run the virtual environment's python interpreter directly using `poetry run`

```bash
poetry run python
```
